🔍 Why z-index: 999 Matters

Without High z-index

Card A
Tooltip → hidden behind Card B
Card B

Tooltip gets clipped by overlapping cards

With z-index: 999

Card A
Tooltip → floats above all
Card B

Tooltip escapes stacking context

📝 JavaScript Implementation

card.addEventListener('mouseenter', () => {
    tooltip.style.zIndex = '999';  // Escape stacking context!
    tooltip.style.display = 'block';
});

card.addEventListener('mouseleave', () => {
    tooltip.style.display = 'none';
});
